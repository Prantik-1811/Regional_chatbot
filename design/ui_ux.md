# UI/UX Design - HK Pilot

## Theme
-   **Colors**: Red (`#EF4444`) and White.
-   **Title**: "HK Cyber Intelligence"

## Components
1.  **Header**: Static title.
2.  **ChatList**: Scrollable area.
3.  **Input**: Text box + Send button.
4.  **SourcePanel**: Accordion under bot messages.

## Flow
1.  User opens app.
2.  Types question.
3.  Bot answers with HK-specific data.
4.  User clicks source link to verify.
